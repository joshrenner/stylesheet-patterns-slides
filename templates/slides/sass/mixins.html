Mixins

Mixins allow you to define styles that can be reused throughout the stylesheet.


@mixin color {
  color: blue;
}

.aMixed {
  @include color;
  width: 200px;
}

.anotherMixed {
  @include color;
  width: 100px;
}


.aMixed {
  color: blue;
  width: 200px;
}
.anotherMixed {
  color: blue;
  width: 100px;
}
